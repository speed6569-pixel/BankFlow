# AI 기능 설계

## 1. 핵심 AI 기능
BankFlow의 핵심 AI 기능은 **금융 상담 챗봇**이다. 사용자의 질문에 따라 소비 패턴 요약, 저축 상품 추천, 소상공인 대출 상담 흐름을 안내한다.

## 2. 왜 이 기능을 선택했는가
1주 프로젝트에서는 AI 기능이 반드시 화면에서 직접 보이는 것이 중요하다. 챗봇은 다음 장점이 있다.

- 사용자가 즉시 체험 가능
- AWS Bedrock 활용 방안을 설명하기 쉬움
- 금융 서비스와 자연스럽게 연결됨
- 데모 시나리오를 짧은 시간 안에 보여주기 좋음

## 3. 현재 구현 방식
현재 웹사이트에서는 모의 응답 방식으로 구현되어 있다.

- 사용자가 질문을 입력
- 키워드 기반으로 응답 분기
- 결과를 채팅 UI에 즉시 표시

이 방식은 실제 Bedrock API가 없어도 AI 기능이 화면에서 동작하는 흐름을 보여줄 수 있다는 장점이 있다.

## 4. 실제 AWS Bedrock 연동 설계

### 사용 리소스
- Amazon Bedrock
- API Gateway
- AWS Lambda
- CloudWatch

### 예상 처리 흐름
1. 프론트엔드에서 사용자 질문 전송
2. API Gateway가 요청 수신
3. Lambda가 사용자 질문과 금융 컨텍스트를 조합해 프롬프트 생성
4. Bedrock 모델이 응답 생성
5. Lambda가 응답을 요약/정제 후 프론트엔드에 반환
6. CloudWatch에 요청 로그 저장

## 5. 시연 가능한 질문 예시
- 이번 달 소비 패턴 요약해줘
- 저축 상품 추천해줘
- 소상공인 대출 상담 흐름 보여줘

## 6. 기대 효과
- 단순 정보 조회보다 더 높은 사용자 경험 제공
- 금융상품 안내를 개인화된 상담처럼 표현 가능
- AWS AI/ML 리소스 활용 능력을 평가 항목에 맞게 설명 가능

## 7. 향후 확장
- 거래 데이터 기반 개인화 추천 고도화
- Bedrock Guardrails 적용
- 상담 기록 저장 및 후속 액션 추천
- 신용평가 보조 모델과 연계한 대출 상담 확장
