# 테스트 보고서

## 1. 테스트 목적
구현한 화면이 실제로 실행되는지 확인하고, MVP 범위에서 주요 동작이 정상적으로 보이는지 점검한다.

## 2. 테스트 대상
- 메인 페이지
- 로그인 페이지
- 대시보드 페이지
- 금융상품 페이지
- AI 챗봇 페이지
- 이체 시뮬레이션 페이지

## 3. 실행한 명령어
```bash
npm install
npm run lint
npm run build
```

## 4. 기능 테스트 항목

| 항목 | 확인 내용 | 기대 결과 |
| --- | --- | --- |
| 메인 페이지 | 주요 섹션 렌더링 | 서비스 개요와 CTA 노출 |
| 로그인 | 입력 UI 및 이동 버튼 확인 | 대시보드 이동 가능 |
| 대시보드 | 자산/소비/거래 렌더링 | 더미 데이터 정상 노출 |
| 금융상품 | 상품 카드 렌더링 | 상품 설명과 특징 표시 |
| AI 챗봇 | 질문 입력 후 응답 생성 | 모의 AI 응답 출력 |
| 이체 시뮬레이션 | 금액 입력 및 제출 | 결과 메시지 출력 |

## 5. 실행 결과

- `npm install`: 성공
- `npm run lint`: 성공, ESLint 경고/오류 없음
- `npm run build`: 성공, 모든 페이지 정적 빌드 완료
