# BankFlow

1주일 안에 완성 가능한 금융/은행 서비스 데모 웹사이트입니다. 실제 코어뱅킹 구현 대신, 평가 기준에 맞는 MVP 화면과 AWS 아키텍처 문서, Bedrock 기반 AI 상담 시나리오를 함께 보여주는 데 집중했습니다.

## 핵심 목표

- **프로젝트 개요 10점**: 1주 기준 현실적인 범위를 문서화
- **AWS 아키텍처 40점**: 리소스 구성과 연결 관계를 명확히 설명
- **AI 리소스 활용 40점**: Bedrock 활용 설계와 실제 UI 데모 제공
- **구현 모듈 안정성 10점**: 로컬 실행 가능한 화면과 테스트 결과 제공

## 구현 범위

### 포함
- 메인 페이지
- 로그인 페이지
- 사용자 대시보드
- 금융상품 소개 페이지
- AI 챗봇 페이지
- 이체 시뮬레이션 페이지
- AWS 아키텍처 및 AI 기능 문서
- 테스트 보고서

### 제외
- 실제 계좌 인증 및 권한 관리
- 실계좌 연동, 결제, 송금
- 실제 Bedrock 호출 및 운영 배포
- 관리자 콘솔, 심사 백오피스

## 기술 스택

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- 더미 데이터 기반 UI 시연

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000` 접속

## 페이지 구성

- `/` 메인 페이지
- `/login` 로그인
- `/dashboard` 사용자 대시보드
- `/products` 금융상품 소개
- `/ai-chat` AI 챗봇
- `/transfer` 이체 시뮬레이션
